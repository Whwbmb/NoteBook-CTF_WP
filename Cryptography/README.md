## 数字签名

|数字签名应具有的性质|
|:--:|
|签名可信|
|签名不可伪造|
|签名不可重用|
|签名文件不可改变|
|签名不可抵赖|

|数字签名两种方法|数字签名方案|

|对消息整体签名|直接数字签名|
|对消息摘要签名|仲裁数字签名|
